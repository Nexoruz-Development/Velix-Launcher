## About

Velix Launcher is a modern Minecraft Java Edition launcher focused on performance, simplicity, and a premium desktop experience.

It combines official Microsoft authentication, automatic Java management, Modrinth integration, powerful instance management, and a clean monochrome interface designed from the ground up.

The project is currently under active development and APIs may change before the first public release.

---

## Features

- Microsoft Authentication (OAuth 2.0 + PKCE)
- Offline Profiles *(ownership verification required)*
- Velix Accounts
- Instance Management
- Modrinth Integration
- Automatic Java Installation
- Java Runtime Management
- Custom JVM Arguments
- Authlib Injector Support
- Theme System
- Fast Startup

More features are planned.

## Authentication

Velix Launcher is designed for legitimate Minecraft players.

Microsoft authentication is required to verify Minecraft ownership before launcher functionality becomes available.

Offline Profiles are available only after ownership has been successfully verified.

Velix Accounts provide launcher-specific features such as cloud synchronization, launcher settings, and cosmetic skin integration. They are **not** Microsoft account replacements and cannot be used to access official Minecraft online-mode servers.
